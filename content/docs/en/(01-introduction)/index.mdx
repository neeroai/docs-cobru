## What Cobru covers

Cobru's public developer surface goes beyond payment creation. The API menu captured from Cobru's own references includes payments, transfers, cash withdrawals, bank withdrawals, digital services, and virtual cards.

### Product areas

| Area | What developers use it for |
|------|-----------------------------|
| Cobrus | Create hosted payment links, calculate fees, inspect payment details, and edit existing payment links |
| Webhooks | Receive asynchronous payment state updates in your own backend |
| Transfers | Move balance between Cobru users |
| Cash withdrawals | Send funds to cash pickup flows such as Efecty |
| Bank withdrawals | Push funds to banks in Colombia and other supported countries |
| Services | Sell digital products such as recharges and PIN-based services |
| Cards | Create and operate virtual cards from your own product workflows |
