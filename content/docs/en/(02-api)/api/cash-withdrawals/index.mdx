Cobru exposes a cash-withdrawal flow under `Retiros en efectivo`.

## Endpoints

| Endpoint | Method | Status |
|----------|--------|--------|
| Withdraw cash | `POST` | documented in legacy materials |
| List withdrawals | `GET` | listed in Cobru API menu |
| Cancel withdrawal | `POST` | listed in Cobru API menu |
| Withdrawal details | `GET` | listed in Cobru API menu |
