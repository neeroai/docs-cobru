## Step 1 — Start the payment

`POST /{url}`

```json
{
  "name": "Test Gómez",
  "payment": "daviplata",
  "cc": "1082626262",
  "email": "test@cobru.co",
  "phone": "307654332",
  "document_type": "CC"
}
```

## Step 2 — Confirm the OTP

Spotlight documents a follow-up endpoint:

`POST /cobru/confirm_daviplata/`

```json
{
  "URL": "ufa8q176",
  "OTP": "123456"
}
```

If everything succeeds, Spotlight describes a response with `result == "OK"`.

<Callout type="warn">
  Cobru has not yet published a full OTP replay for this step in the sandbox evidence
  that backs this repo. Treat the endpoint as a strong legacy contract, not as a
  freshly verified live flow.
</Callout>
