---
title: Credit card
description: Submit cardholder and card fields for a hosted Cobru card payment.
icon: CreditCard
---

## Endpoint

`POST /{url}`

## Request

```json
{
  "name": "Test Gómez",
  "payment": "credit_card",
  "cc": "1082626262",
  "email": "test@cobru.co",
  "phone": "307654332",
  "document_type": "CC",
  "credit_card": "4111111111111111",
  "expiration_date": "12/30",
  "cvv": "123",
  "dues": 1
}
```

## Spotlight notes

- Cobru does not store card data
- production traffic must be HTTPS
- test cards shown by Spotlight:
  - approved: `4111 1111 1111 1111`
  - declined: `4111 1111 1111 1112`
