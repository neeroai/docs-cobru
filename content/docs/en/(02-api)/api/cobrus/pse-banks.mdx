## Endpoint

`GET /get_banks/1/`

<Callout type="info">
  This endpoint appears in legacy Cobru materials and is a common dependency when presenting PSE options to users.
</Callout>

## Example response

```json
[
  { "bankName": "BANCO AGRARIO", "bankCode": "1040" },
  { "bankName": "BANCO AV VILLAS", "bankCode": "1052" },
  { "bankName": "BANCO BBVA COLOMBIA S.A.", "bankCode": "1013" }
]
```
