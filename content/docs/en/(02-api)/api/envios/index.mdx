Cobru's transfer surface appears under `Envios` in the product API menu.

## Endpoints

| Endpoint | Method | Status |
|----------|--------|--------|
| Send money to another user | `POST` | documented in legacy materials |
| List your transfers | `GET` | listed in Cobru API menu, response contract pending verification |
