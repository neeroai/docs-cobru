## Overview

Cobru is a payment platform for Colombia and LATAM. It allows businesses to accept payments via PSE, Nequi, Daviplata, Bre-B (Sistema de Pagos Inmediatos), and credit/debit cards through a single API.

### Key capabilities

| Feature | Description |
|---------|-------------|
| Pasarela de Pagos | Checkout with all local payment methods |
| Cobranza Automatizada | Automated reminders and recurring billing |
| Pagos Masivos | Bulk disbursements with traceability |
| Pagos Internacionales | Cross-border payments and payouts |
| BRE-B | Instant bank transfers via QR (Redeban) |
