Cobru es una plataforma de pagos colombiana para equipos que necesitan crear links de pago, aceptar métodos locales, automatizar cobranzas y operar flujos de movimiento de dinero desde una sola API.

Esta documentación se está reconstruyendo a partir de learnings verificados en sandbox, materiales técnicos históricos de Cobru y la superficie visible en sus propias referencias API. Cuando un contrato aún no está verificado contra una integración real, la página lo indica explícitamente.

## Qué cubre Cobru

La superficie pública para developers va más allá de crear pagos. El menú API capturado desde las referencias de Cobru incluye pagos, envíos, retiros en efectivo, retiros bancarios, servicios digitales y tarjetas virtuales.

### Áreas de producto

| Área | Para qué la usa un developer |
|------|------------------------------|
| Cobrus | Crear links de pago hosted, calcular fees, inspeccionar detalles y editar cobros existentes |
| Webhooks | Recibir cambios de estado de pago en tu propio backend |
| Envíos | Mover saldo entre usuarios Cobru |
| Retiros en efectivo | Enviar fondos a flujos de retiro tipo Efecty |
| Retiros bancarios | Enviar fondos a bancos en Colombia y otros países soportados |
| Servicios | Vender productos digitales como recargas y PINes |
| Tarjetas | Crear y operar tarjetas virtuales desde tus propios workflows |
