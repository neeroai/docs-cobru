## Checklist de plataforma

1. Empieza por [Autenticación](/es/docs/authentication) para estabilizar emisión y caché de tokens.
2. Sigue con [Webhooks](/es/docs/webhooks) antes de tratar el estado de pagos como production-ready.
3. Usa [Pruebas](/es/docs/testing) y [Preparación para producción](/es/docs/production-readiness) para cerrar gaps operativos.
4. Usa [Build with AI](/es/docs/build-with-ai) si tu equipo depende de coding agents o copilots internos.
5. Abre la [Referencia API](/es/docs/api/reference) cuando necesites detalle exacto del contrato.
