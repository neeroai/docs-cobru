La superficie de transferencias aparece bajo `Envios` en el menú API de Cobru.

## Endpoints

| Endpoint | Método | Estado |
|----------|--------|--------|
| enviar dinero a otro usuario | `POST` | documentado en materiales legacy |
| obtener envíos del usuario | `GET` | listado en el menú API de Cobru, contrato pendiente de verificación |
