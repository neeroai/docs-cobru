## Endpoint

`GET /send/`

## Campos vistos en materiales legacy

| Field | Tipo | Notas |
|-------|------|-------|
| `from_user` | `string` | Identificador del emisor. |
| `to_user` | `string` | Identificador del destinatario. |
| `amount` | `number` | Monto del envío. |
| `date` | `string` | Timestamp de creación o fecha formateada del envío. |

## Cómo usar el listado

- reconciliar envíos salientes en un panel operativo
- confirmar si una solicitud de envío ya fue registrada
- construir tooling de soporte para movimientos wallet-to-wallet

## Lo que todavía falta confirmar
