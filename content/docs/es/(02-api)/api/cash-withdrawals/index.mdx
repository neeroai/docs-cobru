title: Retiros en efectivo
description: Retira saldo Cobru a través de flujos de retiro en efectivo como Efecty.

---

Cobru expone una familia de retiros en efectivo bajo `Retiros en efectivo`.

## Endpoints

| Endpoint | Método | Estado |
|----------|--------|--------|
| retirar dinero por efectivo | `POST` | documentado en materiales legacy |
| obtener retiros | `GET` | listado en el menú API de Cobru |
| cancelar retiro | `POST` | listado en el menú API de Cobru |
| detalles del retiro | `GET` | listado en el menú API de Cobru |
