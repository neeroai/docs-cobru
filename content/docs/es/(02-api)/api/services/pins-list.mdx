---
title: Lista de PINes
description: Recupera el catálogo de productos digitales tipo PIN disponibles en Cobru.
icon: List
---

<Callout type="warn">
  Este endpoint de catálogo está documentado desde exports legacy de Cobru. Verifica filtros y
  schema del catálogo en un ambiente real antes de depender de él.
</Callout>

## Endpoint

`GET /pines/all_pines/`

## Variante opcional de query vista en docs antiguas

`GET /pines/all_pines/?operator={operator}`

Usa la variante filtrada por operador cuando quieras prefiltrar el catálogo de PINes para un
proveedor específico.
