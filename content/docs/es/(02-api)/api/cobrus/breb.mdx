## Endpoint

`POST /{url}`

## Request

```json
{
  "payment": "breb"
}
```

Esta es la variante más mínima de `payment-details` documentada en Spotlight.

<Callout type="warn">
  Spotlight documenta `payment: "breb"` directo, pero el research posterior en
  sandbox observó esta ruta como deshabilitada o restringida comercialmente en
  algunos entornos. Confirma la disponibilidad directa de Bre-B en tu propio sandbox
  antes de tratarla como lista para producción.
</Callout>

## Qué esperar en la respuesta

Spotlight muestra una respuesta de `paymentorder` que incluye:

- `qr_value`
- `key_value`
- `method: "breb"`
- `franchise: "breb"`

Los materiales originales también indican que `qr_value` y `key_value` expiran en 5 minutos.

## Cuándo usarlo

Usa Bre-B cuando quieras una opción de transferencia interoperable y tu producto pueda trabajar con instrucciones de pago de vida corta más confirmación por webhook.
