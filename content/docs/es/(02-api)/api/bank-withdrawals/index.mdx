El menú API de Cobru expone una superficie amplia de retiros bancarios para Colombia y otros países.

## Cobertura visible hoy

| Operación | Estado |
|-----------|--------|
| lista de bancos disponibles | listado en el menú Cobru |
| tus retiros | listado en el menú Cobru |
| crear retiro a Colombia | listado en el menú Cobru |
| detalle del retiro | listado en el menú Cobru |
| crear retiro a Brasil | listado en el menú Cobru |
| crear retiro a México | listado en el menú Cobru |
| crear retiro a Argentina | listado en el menú Cobru |
| transferencia BRE-B | listado en el menú Cobru |

<Callout type="warn">
  La superficie de producto es clara, pero la mayoría de contratos request/response de esta sección siguen pendientes de extracción real o de un export actualizado de Stoplight.
</Callout>
