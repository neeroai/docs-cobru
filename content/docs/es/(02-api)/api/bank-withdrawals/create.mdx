## Qué debes confirmar con Cobru

- path exacto y firma del método
- campos de identidad del beneficiario
- nombre del campo para código de banco
- restricciones de tipo de cuenta
- modelo de settlement síncrono vs asíncrono
- comportamiento de callback o webhook para cambios de estado
