---
title: Congelar tarjeta
description: Congela o bloquea una tarjeta virtual de Cobru.
icon: Snowflake
---

<Callout type="warn">
  Congelar y descongelar está documentado desde exports legacy de Cobru. Confirma
  reversibilidad y efecto sobre autorizaciones pendientes antes de exponer esta función al
  usuario final.
</Callout>

## Endpoint

`POST /cobru/debit/freeze_card/`

## Campos del request vistos en documentación legacy

| Field | Tipo | Requerido | Notas |
|-------|------|-----------|-------|
| `card_id` | `number` | sí | Identificador de la tarjeta. |
| `freeze` | `boolean` | sí | `true` para congelar, `false` para descongelar. |
