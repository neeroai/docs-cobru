El menú API de Cobru incluye una sección `Tarjetas`, lo que sugiere una capacidad más amplia de tarjetas programables.

## Operaciones visibles

| Operación | Estado |
|-----------|--------|
| crear tarjeta | listado en el menú Cobru |
| recargar tarjeta | listado en el menú Cobru |
| detalles y listado | listado en el menú Cobru |
| movimientos | listado en el menú Cobru |
| congelar tarjeta | listado en el menú Cobru |

<Callout type="warn">
  Estas páginas documentan la superficie visible actual y las preguntas de integración, no un contrato de tarjetas ya verificado extremo a extremo.
</Callout>
