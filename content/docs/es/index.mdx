## Resumen

Cobru es una plataforma de pagos para Colombia y LATAM. Permite a los negocios aceptar pagos via PSE, Nequi, Daviplata, Bre-B (Sistema de Pagos Inmediatos) y tarjetas débito/crédito a través de una sola API.

### Capacidades principales

| Funcionalidad | Descripción |
|---------------|-------------|
| Pasarela de Pagos | Checkout con todos los métodos de pago locales |
| Cobranza Automatizada | Recordatorios automáticos y cobro recurrente |
| Pagos Masivos | Dispersión masiva con trazabilidad |
| Pagos Internacionales | Pagos y cobros internacionales |
| BRE-B | Transferencias bancarias inmediatas via QR (Redeban) |
