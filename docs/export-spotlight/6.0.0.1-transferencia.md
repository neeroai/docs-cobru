## Generar retiros por Bre-B

El endpoint es el siguiente:


```json
    URL: https://dev.cobru.co/withdraw/co/
    Method: POST
    Content-Type: application/json
```
Debemos enviar la siguiente información:

|Key| Tipo de dato| Descripción|
|----------|----------|----------|
|amount|Int|Monto a retirar en pesos colombianos (COP).|
|callback|String|URL para recibir notificaciones vía webhook del estado del retiro. (opcional)|
|description|String|Descripción o referencia del retiro. (opcional)|
|account_holder_name|String|Nombre del titular asociado.|
|key_value|String|LLave del destino (ej.: número de celular o alias, como "@LLAVE3065").|

Ejemplo 

``` json
{
    "amount": 2000,
    "callback": "https://webhook.site/sdjskdskjd-f95c-4b9a-9771-9525c86a0b4f",
    "description": "retiro de breb",
    "account_holder_name": "Ed Test",
    "account_type": 8,
    "key_value": "@LLAVE7984"
}

```
