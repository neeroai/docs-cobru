# Obtener tu Balance

El endpoint para obtener tu balance: 
```json
let response = await fetch("https://dev.cobru.co/balance/", {
        method: 'GET',
        headers: {
            Accept: 'application/json',
            'Content-Type': 'application/json',
            'Authorization': "Bearer TU_ACCESS_TOKEN",
            'x-api-key' : "TU_LLAVE_PUBLICA"
        },
    });
```
si todo esta ok, recibiras el balance con un status 200, de lo contrario un error
