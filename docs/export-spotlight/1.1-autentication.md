## Autenticación

Para utilizar el API, necesitarás dos elementos esenciales: la API key y un token de acceso. 

- **API key**: Requerida para acceder de forma general a la API de Cobru.
- **Token de acceso**: Necesario para realizar operaciones específicas usando tu cuenta.

Estas credenciales deben incluirse en los headers de tus solicitudes al API, en las siguientes claves:

- `Authorization`
- `x-api-key`

### Ejemplo de consulta de balance

A continuación, se muestra cómo realizar una consulta de balance utilizando JavaScript:

```javascript
// Un simple GET con el header de Authorization y su respectivo Token
let response = await fetch("https://dev.cobru.co/balance/", {
    method: 'GET',
    headers: {
        Accept: 'application/json',
        'Content-Type': 'application/json',
        'Authorization': "Bearer TU_ACCESS_TOKEN",
        'x-api-key': "TU_LLAVE_PUBLICA"
    },
});
```

> **Nota:** Asegúrate de incluir un espacio entre la palabra `Bearer` y el token de acceso en el header `Authorization`.

### ¿Cómo accedo a la API de Cobru?

Para comenzar tu integración, visita la app de Cobru y dirígete a:

`Más -> Integración`

Ahí encontrarás tu `x-api-key` y el `refresh token`.

#### Renovación del token de acceso

Una vez tengas tu `refresh token`, utiliza el siguiente endpoint para obtener un nuevo token de acceso:

```http
URL: https://dev.cobru.co/token/refresh/
METHOD: POST
Content-Type: application/json
```

El cuerpo de tu solicitud debe ser un objeto JSON convertido a texto (`application/json`) con las siguientes propiedades:

| Parámetro | Tipo   | Descripción          |
| --------- | ------ | -------------------- |
| refresh   | String | El refresh token     |

#### Respuesta esperada

La respuesta incluirá los siguientes parámetros:

| Parámetro | Tipo   | Descripción                                   |
| --------- | ------ | --------------------------------------------- |
| access    | String | Token de acceso válido por 15 minutos         |

#### Ejemplo en JavaScript

```javascript
let response = await fetch("https://dev.cobru.co/token/refresh/", {
    method: 'POST',
    headers: {
        Accept: 'application/json',
        'Content-Type': 'application/json',
        'x-api-key': "asdfghjkl1234456"
    },
    body: JSON.stringify({
        refresh: "TU_REFRESH_TOKEN",
    }),
});

if (response.status === 200) {
    console.log(await response.json());
}

// La respuesta en consola sería:
{
    "access": "TU_NUEVO_ACCESS_TOKEN"
}
```

> **Nota:** El token recibido será válido por 15 minutos. Pasado este tiempo, deberás renovarlo usando el mismo endpoint.
